Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost

Authenticating requests

This API is not authenticated.

Tournament Management

APIS for managing tournaments

Get tournaments

Tournament list is ordered by creation time.

Example request:
curl --request GET \
    --get "http://localhost/tournaments?page=1&limit=10" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/tournaments"
);

const params = {
    "page": "1",
    "limit": "10",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->get(
    'http://localhost/tournaments',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'query' => [
            'page'=> '1',
            'limit'=> '10',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
import requests
import json

url = 'http://localhost/tournaments'
params = {
  'page': '1',
  'limit': '10',
}
headers = {
  'Content-Type': 'application/json',
  'Accept': 'application/json'
}

response = requests.request('GET', url, headers=headers, params=params)
response.json()

            "updated_at": "1970-01-01 00:00:00",
            "_url": "/tournaments/1"
        }
    ]
}

Request      

GET tournaments

Query Parameters

page  integer optional  

Defaults to 1. Values less than 1 will default to 1.

name  string optional  

Search for tournament that contains name keyword.

limit  integer optional  

Defaults to 10. Values less than 1 will default to 1.

Response

Response Fields

count  integer  

total_pages  integer  

next_page  string  

Returns null if last page is reached.

prev_page  string  

Returns null if it is the first page.

results  object[]  

List of tournaments

results.id  integer  

Tournament ID.

results.name  string  

Tournament name.

results.description  string  

Tournament description.

results.created_at  string  

Tournament creation timestamp.

results.updated_at  string  

Last update timestamp.

Create a tournament

Example request:
curl --request POST \
    "http://localhost/tournaments" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"Tournament Awesome\",
    \"description\": \"Lorem ipsum dolor sit...\"
}"

const url = new URL(
    "http://localhost/tournaments"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "Tournament Awesome",
    "description": "Lorem ipsum dolor sit..."
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost/tournaments',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'name' => 'Tournament Awesome',
            'description' => 'Lorem ipsum dolor sit...',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
import requests
import json

url = 'http://localhost/tournaments'
payload = {
    "name": "Tournament Awesome",
    "description": "Lorem ipsum dolor sit..."
}
headers = {
  'Content-Type': 'application/json',
  'Accept': 'application/json'
}

response = requests.request('POST', url, headers=headers, json=payload)
response.json()

Example response (201, Created):


{
    "id": 1,
    "name": "Tournament 1",
    "description": "The mind is willing, but the body is spongy and bruised",
    "created_at": "1970-01-01 00:00:00",
    "updated_at": "1970-01-01 00:00:00",
    "_url": "/tournaments/1"
}

Example response (422, Invalid Request Body):


{
    "message": "Please give a name to the tournament (and 1 more error)",
    "errors": {
        "name": [
            "Please give a name to the tournament"
        ],
        "description": [
            "Description must be under 500 characters"
        ]
    }
}
Received response:
Request failed with error:

Request      

POST tournaments

Body Parameters

name  string  

Name of the tournament. Must be less than or equal to 100 characters. Must not be greater than 100 characters.

description  string optional  

Short description of the tournament. Must be less than or equal to 500 characters. Note: WYSIWYG tags and markdowns are considered too. Must not be greater than 500 characters.
